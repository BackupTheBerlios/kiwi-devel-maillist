Hi,

Attached is a patch to support a new element <vmconfig-entry>. This
element allows the user to specify test that ends up in the .vmx or
xenconfig file.

Certain default options currently written by Kiwi may be over written by
these settings.

Documentation has been updated to reflect the changes. Also I noticed
the documentation for the <machine> element was not quite correct, nor
consistent with documentation for other elements w.r.t. presentation. I
attempted to address this problem.

Last but not least I noticed that I neglected to update the docs when I
added the new post dump behavior options recently :( , thus this patch
addresses this problem and the new options and behavior is now
documented. :)

Enjoy,
