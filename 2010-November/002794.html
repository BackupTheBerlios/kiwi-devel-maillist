The options (vmnic, vmdvd,vmdisk) that can be configured for a virtual machine are very limited right now. If another sub-element like vmxoption is enabled for <machine>, the users will be able to configure a lot of options for the Virtual machine.

For example, the users can add
<machine  memory="1024" guestOS="suse">

<vmxoption sched.mem.max="4GB">

</machine>

All that has to be done is, to copy the option to vmx file along with its value. I believe adding such an option will add a lot of flexibility for the users to configure their VMs.

What do you guys think?
