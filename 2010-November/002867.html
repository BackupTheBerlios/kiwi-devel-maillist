Hi,

The attached patch eliminates the use of getImageTypeAndAttributes from
the KIWIImageFormat format object.

This also required changes in Image.pm and Boot.pm as we create XML
objects there and the build settings needed to be updated after the
creation of new XML objects. Longer term I am thinking we should be able
to get away with having the XML object created only in a build manager
type of object.

We'll see where that idea leads. ;) One step at a time, XML processing
first.
