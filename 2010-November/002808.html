> We are using a private instance of OBS and kiwi, to compose some appliances. Because of these missing extra options, the user has to download the image and manually add/change some of the extra VMX options. I was hoping to eliminate these manual steps.
> 
> 
> Of course, I understand that it is not possible to check all the options VMX supports and I agree that it has to be well documented to let the user know that he/she will be using this option at his/her own risk.

Robert provided a patch which allows to set that up as follows:

   <machine>
      ...
      <vmconfig-entry>text</vmconfig-entry>
   </machine>

you can have multiple vmconfig-entry elements in your machine section
