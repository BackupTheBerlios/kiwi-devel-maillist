Hi All,

I would like to introduce a XSD scheme for the kiwi config.xml
file. The format of the file seems to be somewhat stable and it
makes sense to check it more detailed than this is done right
now.

With the scheme definition there is a small change in the config.xml
description as well. The specification of the size value was set to:

	<size unit="M">60</size>

I was told this is not common and should be changed to:

	<size unit="M" value="60"/>

I would do that for all images in the repository. If you are using
your own description please change it.

Comments, suggestions are welcome :-)
I will submit the patch the next days if there are no objections

Thanks
