Changes Report kiwi v4.73 2011-01-30 - 2011-02-04
--------------------------------------------------
most important changes for this weeks release.
Next week I will take a look into image caching again
some ideas came to my mind so let's see if I can improve
this code a bit

-------------------------------
Features
-------------------------------
commit 1ce6fcfa13512c8df05f2e7b9f77fecd8981b2c0
commit 5d9dcadd9a51386791805a22edffb616b95baec1

    - improved ec2 format creation per request in #667303
    - DB: update ec2 docs and examples per request in #667303
      https://bugzilla.novell.com/show_bug.cgi?id=667303

commit 4902e377e83ddea6e3ad6a321b56c2f9e003720f

    - fixed ec2 examples

commit f719e7094271c687a1fd09e2bb78dacaa80d042a

    - added support for plain:// url types. They forward the URL
      (everything following "plain://") unmodified to the package
      manager. This can be used if kiwi does not support a special
      URL but the package manager does

commit e4e5cff57485642cb691b378ea71c0048570702b

    - added support for username and password in repository elements
    - enhanced cifs mount (smb://) to be able to use credentials

-------------------------------
Architecture (s390)
-------------------------------
commit 86f58c043fd672d2d8c0b599d68df5b86f1a64c4

    - update s390 boot code according to generic changes in the x86 boot code

-------------------------------
Profiles
-------------------------------
commit 2e954a10b3e6357c0f529f01fbca0cc2216be161

    - fixed use of profiled repository sections
      something like this now works:

      <repository ... profile="http">
         <source path="some-http-repo"
      </repository>
      <repository ... profile="obs">
         <source path="some-obs-repo"
      </repository>
