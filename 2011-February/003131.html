Changes Report kiwi v4.79 2011-02-20 - 2011-02-25
--------------------------------------------------
most important changes for this weeks release. Due to the
11.3 release process there were several kiwi releases too
so don't wonder. The Week ended with 4.79 and if everything
goes fine this is the final release for kiwi on openSUSE 11.4

Until 11.4 is out we will only accept bugfixes and close
master for further development.

-------------------------------
P1 bugs
-------------------------------
commit 7430026df02f537711186be3200a7675c02892a4

    - The default tftp block size chosen by kiwi is too small and
      not aligned to page size (4096), so it causes unnecessary
      overhead. This patch improves loading performance a lot

commit e4b97d7c153df06659f14b8be7b8a80ffc635f8d

    - only compare the major version of squashfs kernel module
      and squashfs tools (bnc #674653)

commit 4d10d513c012bd9885d59d08236c37c0fd53e4c5

    - fixed cleanup of shared memory segments and semaphores
      by a valid destructor in KIWISharedMem.pm

commit 704c00971b5b5972e0bfa67dc584db81b18195b9

    - fixed regexp in __checkPatternTypeAttrConsistent()

commit e40ada440844af10bd82a8582188115a8898b59e

    - setup owner/group permissions only if the user home
      directory exists (bnc #658172)


-------------------------------
Unit tests
-------------------------------
commit e52b2cf1ec4a8ec68f9a1f93d5e361f8f9537cf4

    - added unit testing for the KIWILocator object

commit 4c1bffd51d839986245b291871db9b5c3fb89206

    - added check to enforce the "no whitespace" rule for profile names

commit 4f593b22118e24d1feb2feef363914b8f23ab1f7

    - enhanced patternType check which now allows the use of the patternType
      attribute for all <packages> specifications that add packages to the
      image


-------------------------------
Late Features
-------------------------------
commit 3fa63f6f8ae98a5960df3b5d09777604e6b0dfa1

    - add optional support for hybrid iso for product medias

commit 171c1d5a677eeafcc542015699eaaca90045ba93

    - added support for config-cdroot.tgz and config-cdroot.sh in
      OEM install media (CD and stick) (bnc #673566)

commit 11f3f8eb9973c61ddbff5d13f53360ce215cae56

    - moved check for ec2-bundle-image after the check for the
      Amazon AWS credentials

commit 8fb3d4b63fdcda447fcb0ab2901ee73480aa4508

    - added support for netboot config files update. The update
      happens according to the value of an optional md5sum hash
      specified per file
