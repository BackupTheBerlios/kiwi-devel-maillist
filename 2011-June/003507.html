Hi All,

I'm currently adding templates for openSUSE 12.1 and while
doing that I found that we have templates for some older
distributions in the code. So far I think they are not
maintained very well and I would like to reduce the amount
of example template descriptions to something which
we can maintain

* Package kiwi-doc, provides: /usr/share/doc/packages/kiwi/examples
  This directory contains examples for 11.2, 11.3, 11.4 and 12.1
  Is anybody using these examples ? I'd like to change this to
  the last released version. At the moment: '11.4' Any objections ?

* Next are the amount of examples in for example
  /usr/share/doc/packages/kiwi/examples/suse-11.4. We have
  examples for each kiwi image type and we also document them
  in the kiwi docs. But we also have additional examples like
  suse-euca-cloud or suse-min-gnome. I'd like to remove them
  here and let them be part of the wiki where we reference
  them. Robert this is mainly a question for you ? Any objections ?

* Next question refers to the enterprise products (SLE). At the
  moment we distinguish between SLES and SLED. I would like to remove
  SLED as I think it doesn't make sense. System Imaging's target
  is definitly the server not the desktop. The description itself
  just differs in terms of repos and packages. Any objections ?

* Package kiwi-templates, provides JeOS examples for 11.2, 11.3
  11.4, 12.1 SLE10 and SLE11. I would like to reduce them to
  SLE10, SLE11 and the two latest released openSUSE versions (11.3, 11.4)
  at the moment. Any objections ?

Please speak up now if you see your examples to vanish or if
you see other problems. Let's find the best solution here

Thanks

Regards,
Marcus
