Changes Report kiwi v4.91 2011-05-29 - 2011-06-03
--------------------------------------------------
most important changes for this weeks release

---------------------------------------
Features
---------------------------------------
commit a64ea0a4ce0370f1b428eb748043c57aac0f8ff4

    - added option --start-sector to kiwi. This allows to influence
      the default start sector 2048 to be another value e.g. 32 for
      the old style disk layout.

---------------------------------------
P1 bugs
---------------------------------------
commit 093c6b54c2676dbf1e24209d92f819f3d4575cd1

    - allow username and password to be transfered via
      updateDescription / __updateDescriptionFromChangeSet

commit 79693e6601d8b634239929f13bfb0c3b07ccb893

    - fixed call of cleanup functions for KIWIBoot objects
      This has left over an active loop mount under certain circumstances

commit d016ccbb93daa697438085f52cb6ee3abf5cb586

    - really fixed setupConsole to remove the ttyS0 console if no
      ttyS0 device exists (bnc #695948)

commit fb3393a566f5074b5eec3178d21914120729dc79

    - fixed build: Added zypper and filesystem utils to BuildRequires.
      This is needed in order to make the unit test run happy

Marcus
