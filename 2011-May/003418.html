Hi,

The attached patch addresses the ownership change issue discovered by 
EMC. The underlying problem is that tar in verbose mode inserts padding 
to get a pretty table layout. This causes shift of fields and therefore 
causes out file list of files in the archive to be corrupt. This patch 
eliminates the padding problem by using a short listing and a different 
grep command to eliminate the directory entry.
