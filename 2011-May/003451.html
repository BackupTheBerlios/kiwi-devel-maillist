Hi,

The attached patches get the unit tests working again. The main test 
code now uses the new globals object. Also, the 0005 patch adds 
arguments to ImageCreator::createBootImage which allows us to eliminate 
*Initrd* information storage on teh COmmandLine object. The *Initrd* 
info should not be stored on the CommandLine Object as this information 
cannot be set on the kiwi command line.

Some extraneous members remain on the CommandLine object, these will be 
addressed shortly.
