Hello Community

KIWI-LTSP, a LTSP5 implementation on openSUSE using KIWI imaging is
now available. The improvements from the last release has mostly been
in KIWI netboot image.

- Netboot image now uses udev
- The image uses new "diskless" profile helping reduce its size
- Clients with 128MB ram should now be able to boot up properly

All the features as mentioned earlier remains the same
(NBD+aufs/NFS+bind mount root, LDM2, auto mount of local disk, local
sound)

Thanks Schaefi for the great work on getting all improvements we
required in KIWI.

There is a very nice overview of KIWI on Novell Open Audio this week,
check it out to know what else can this technology do:
http://www.novell.com/feeds/openaudio/?p=185

Details on how to set everything up is available here:
http://en.opensuse.org/LTSP

Developers can check out the SVN and other details here:
http://developer.novell.com/wiki/index.php/KIWI-LTSP

Have a lot of fun!

-J
