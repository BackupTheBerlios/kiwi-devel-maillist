You just need standard 4.7G 10.3 i586 DVD. If you are using some other
media, you would have to find all the missing packages and copy them
in: /usr/share/kiwi/image/ltsp/suse-10.3/extra-packages/

All missing packages means all the failed/package not found/file not
found errors you will see in kiwi log.
