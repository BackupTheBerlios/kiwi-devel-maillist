Hei List,

With the checking #1528 the kiwi-instsource module code is usable for the 
openSUSE-11.0 FTP tree. There were some issues with metadata which are 
resolved now. I created an FTP repository _and_ an iso file (which is ~26G) 
from which I installed vmware instances successfully.

The update during installation also works now.

Please note that the metadata may change according to 
http://en.opensuse.org/Product_Management/Code11/installed

I plan to put the specific code into plugins as soon as my time allows.
I also plan to run the iso generation plugin based. At the moment this is done 
manually using the suse-isolinux script provided by kiwi-desc-isoboot.

The package kiwi-instsource will probably require that one as soon as I finish 
the plugins.

For the near future we'll work on integration to the OBS first.
