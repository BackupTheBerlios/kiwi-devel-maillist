Hi all,

v1.49 is building. The following is a list of bugfixes we found
after 1.47 was out, sorry for any inconvenience

v1.48
  - fixed pattern support for local repositories. if the repo-type yast2
    is used the wrong local path is used by kiwi

v1.49
  - added setValidateRepositoryType() method. if you use yast2 as repo-type
    and the URL type opensuse://... kiwi will automatically use the rpm-md
    repotype because all openSUSE buildservice repos are no yast2 repos.
    But the user can set its own repo by using the option --set-repo and in
    this case you don't want the repo-type adaption.

  - fixed creation of stick image, it might happen that there is no space
    left to copy the grub stages or the grub message file

  - fixed mountSystem() in case of squashfs/aufs mounts to use
    eventually existing write extend instead of creating a new one
    all the time which overwrites the previously stored data.

  - fix by Jan-Christoph Bornschlegel <jcbornschlegel at novell.com>
    extraction of message file from initrd doesn't work in case of
    relative paths

Have Fun :-)

Regards,
