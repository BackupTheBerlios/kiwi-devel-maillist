Hi all,

v1.47 is building. The following is a list of bugfixes and
enhancements:

Bugzilla:
=========
   - added check for squashfs in iso creation (#289552). If there
     is no mksquashfs available but the flags=unified is set it is
     ignored and the old live media system is used
   - added special handling in probeDevices() for ata_piix (#292782)
   
Enhancements:
=============
   * added suseGFXBoot function in KIWIConfig.sh. This will activate
     the graphical boot screen not only for live DVD but also for
     the other image types. As side effect the kernel framebuffer is
     now activated. By default the mode 0x318 is used (1024x768).
       
   * compress RW part of an old-style live CD/DVD to enable fast boot.
     If the old system is used kiwi generates a RW image which is loaded
     into RAM. This image wasn't compressed up to now which means the
     complete RW data must be read from the CD/DVD in order to get written
     into RAM. To speedup this process the RW part will be compressed now
     and decompressed during the read process. The amount of data which
     needs to be read from CD/DVD is much smaller and decompression on the
     fly only depends on the processor speed which is faster than the I/O
     performance of a CD/DVD drive

Other Bugfixes:
===============
   - fixed KIWIMigrate, creates activation files in wrong place
   
Have fun :-)
