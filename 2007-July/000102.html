Hi all,

v1.42 is building. The following is a list of bugfixes and
enhancements:

Bugzilla:
=========
  - don't use fstab from host system, create default file (#286298)
  - fixed isoboot theme used for SLED (#288849)

Enhancements:
=============
  - allow --add-package in --upgrade mode. This will install additional
    packages as specified into the existing root tree after the upgrade
    was finished successfully

  - added support for parted partitioner in KIWILinuxRC.sh
    parted is used in sector-size mode which allows partitions of
    uncommon sizes (not aligned). This is useful for compressed
    read-only image parts which means a waste of space if the partition
    is bigger than the image size. Activate the use of parted by
    setting the following in config.<MAC>:

        PARTITIONER=parted

  - added optional config.<mac> parameter KIWI_KERNEL. Up to now only
    the initrd can be downloaded, with KIWI_KERNEL also the kernel
    file can be downloaded. This allows you to remove the /boot/* from
    your system image.
    
  - added vmware package/configuration description section. This
    will lead to an appropriate .vmx file for use in vmware player.
    All you need to do is set up the following section in config.xml
 
    <packages type="vmware" memory="512" disk="ide0">
    </packages>
 
    ---> optional you can add packages you want to have in case of
         vmware. They get installed automatically if you call the
         create step. But be aware once installed they will not be
         removed from the tree

  - snorp added a configuration method named baseStripLocales which
    removed all unused locales except the ones you specify as parameter
    to this function. Very useful for small images. Use this function
    in config.sh

Other Bugs:
===========
  - fixed standalone grub setup if UNIONFS_CONFIG is used    
  
Have fun :-)
