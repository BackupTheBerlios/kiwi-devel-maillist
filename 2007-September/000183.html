It really is very simple, you just have to add the correct
installation source and a bunch of rpms that are on on installation
media in one folder and add that to two config.xml. This part is most
critical. That is why a retail DVD or online ftp inst-source would be
ideal.

So in short, just adding correct installation source and one line on
the top of config.xml is all that is required to set everything up.
After that it is just one command- setup-ltsp.sh that sets each and
everything up.

We are working on implementation that should be even simpler. Test
packages are in home:cyberorg:experimental, it is only for openSUSE
10.3.
