Hi All,

v1.63 is building. The following is a list of
bugfixes and enhancements:

Bugzilla:
=========
   - added usbboot boot image for SLED10 (#308187)
   - added bootsplash support for netboot (#308233)
     implement option --setup-grub-splash
     
Enhancements:
=============
   - The option --setup-grub-splash can be used to integrate the
     splash files of the boot image (initrd) to a format readable
     by the kernel and therefore useful for grub and other boot
     loaders except isolinux. The result of the call is .splash.gz
     file in addition to the "normal" boot image .gz file.
     Together with the config.xml type named "pxe" the splash
     initrd is created automatically.

Other Bugfixes:
===============
   - added bootsplash packages by default for netboot
     and usbboot boot descriptions

Have Fun :-)
