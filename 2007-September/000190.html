Hello Community

If you have 10.3 Beta 2 DVD, install kiwi-desc-ltsp from here:

http://download.opensuse.org/repositories/home:/cyberorg:/experimental/openSUSE_Factory/noarch/

cd /usr/share/kiwi/image/ltsp/suse-10.3

Edit kiwi-ltsp-setup.conf file(hopefully we will have YaST module
soon) and put in correct variables according to your setup. After you
are done with that, run:

sh setup-ltsp -s

This will take a while depending on the speed of your comupter, once
the script completes, you should be able to pxe boot clients.

The script sets up squashfs image and all services you require such as
DHCP, NBD, TFTP to run LTSP.

Please note that no other config file needs to be edited, you do not
need any other extra package than those on 10.3 DVD, a result of
feedback from testers like Jeff on beryl-devel list.

Please do test, all kinds of input greatly appreciated.
