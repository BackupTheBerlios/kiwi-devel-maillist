Magnus and I have a wrapper script that we use for kiwi-ltsp, which is
in three parts, conf, functions and script. I have adapted it slightly
for kiwi in general. It is not 100% functional yet, it can do only
netboot image successfully on 10.3 Beta2, tested with
kwliveDVD-suse-10.3 config which did not work.

I have not played with xen, vmx, usb, so some help from anyone who has
would be great.


Attached are the three files:

kiwi-tool (goes in /usr/bin)
kiwi-tool.conf (/etc/kiwi-tool.conf)
kiwi-tool-functions.sh (/usr/share/kiwi/tools/kiwi-tool-functions.sh)

Change the /etc/kiwi-tool.conf and run kiwi-tool, it will give the usage help.
