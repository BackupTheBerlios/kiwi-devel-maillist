Hi Fruit Lovers ;)

v1.64 is building. The following is a list of
bugfixes and enhancements:

Enhancements:
=============
 - make use of the profile mode by James Willcox and added a default
   and a xen profile into the netboot boot images. According to this
   the old xennetboot boot images has been removed and are replaced
   by the new netboot boot image descriptions. To distinguish between
   a netboot with default or Xen kernel a new type attribute called
   
		bootprofile=...
	
   has been implemented. By default the attribute will
   have the value "default" but this can be overwritten with
   another bootprofile attribute of a system image with "pxe"
   image type. For example: 

        Instead of boot=xennetboot/suse-...   write
                   boot=netboot/suse-... bootprofile="xen"

   Thanks James for the profile mode, that's really a nice feature :)

Other Bugfixes:
===============
   - changed oemboot repartition code to use the same default
     partition names than netboot does. This means the partition
     are as follows: root -> xda2 / swap -> xda1 / home -> xda3
   - changed KIWIPattern.pm to know about compressed patterns. With 10.3
     the yast2 repositories provides compressed meta and pattern files

Have Fun :-)
