Hello Community

Along with the release of openSUSE 10.3 RC1, I am happy to announce
the release of KIWI-LTSP 0.3.8.

Changes from the last release:

* Better X detection thanks to LTSP5's configure_x.sh script
* Sound now auto-detected and works without any manual configuration
(thanks snorp)
* Local disks are now auto-mounted in /home/$USER/ltspfsmount, not a
clean implementation as it still requires manual unmount.
* YaST can now be used for initial image creation and all required server setup.
* Requires openSUSE 10.3 RC1 i386 DVD iso/media.
* 1-click installation of kiwi-ltsp-desc and other required packages.


For a step-by-step howto see: http://en.opensuse.org/LTSP

And last but not the least, we were on the news: http://news.opensuse.org/?p=263

Have a lot of fun!

-J
