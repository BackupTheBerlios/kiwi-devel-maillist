Hello Community

Following the release of openSUSE 10.3 and many new improvements to
kiwi 1.63, kiwi-ltsp package has been updated to work with those.

Please install smart package from
/usr/share/kiwi/image/ltsp/suse-10.3/extra-packages folder after you
install kiwi-desc-ltsp.

Here are the changes from the previous version:

* Config option in kiwi-ltsp-setup.conf to setup NBD or NFS root image
and set up all the required services accordingly.

* Couple of more switches to setup-ltsp, now all the switches are as below:

setup-ltsp.sh

   -p    - Perform a pre-check. This will check that mandatory
packages are installed
   -s    - This server is a dedicated LTSP server. Build it all and
enable services
   -d    - Sets up a /etc/dhcpd.conf.ltsp which can replace dhcpd.conf
   -t    - Sets up tftpboot KIWI/config.default
   -a    - Build the LTSP server and create a boot image that clients
can retrive via PXE
   -l    - Build the LTSP server, just the chroot image
   -n    - Build the KIWI netboot boot image
   -n1   - Prepare the KIWI netboot initrd image (Note! This is a
developer switch)
   -n2   - Finalize the KIWI netboot initrd image (Note! This is a
developer switch)

The package can be found here:

http://download.opensuse.org/repositories/home:/cyberorg:/experimental/openSUSE_Factory/noarch/

Have fun!
