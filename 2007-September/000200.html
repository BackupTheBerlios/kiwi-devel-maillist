Hi,

v1.62 is building. This is a bugfix release mainly for
users who create openSUSE 10.3 images

Bugfixes:
=========
    - fixed changed location of squashfs module in 10.3 kernel
      unified live DVDs will work again with this patch
    - added --auto-agree-with-licenses option for zypper
      prevent license questions
    - include packagemanager adaption from system to boot image
      if the system image uses zypper as packagemanager this
      packagemanager will be used for dependant boot image
      builds

Have Fun :-)
