Hello Community

Let me wish you all a very happy New Year.

I am happy to announce another development release of KIWI-LTSP - a
LTSP5 implementation for openSUSE using KIWI imaging technology.

The improvements from the last versions are:

1. Slimmer netboot image.
2. Installation source checking from captain_magnus

Here are the three steps to get you all set up.

1. Click here to start the installation of all the required packages:

http://download.opensuse.org/repositories/server:/ltsp/openSUSE_10.3/kiwi-ltsp.ymp

2. Edit /etc/sysconfig/kiwi-ltsp either using your prefered text
editor or GUI YaST2 -> /etc/sysconfig -> Kiwi-ltsp

3. Once the installation source path and the IP addresses are entered
correctly, run the following as root in shell:

kiwi-ltsp-setup -s

You can now boot up the client to test.

Have a lot of fun!
