Hi,

> I started the image with an VM (the first time I used a HP server with virtual media).
> I get the following error on console 3
>  mount -t squashfs /dev/loop1 /read
> mount: unknown filesystem type 'squashfs'
> 
> I checked the iso file (config.isoclient is present)
> 
> I guess, SLES10 SP2 doesn' t have support for squashfs, I' ll try cromfs.

neither cromfs nor squashfs is by default part of SLE10. squashfs works
reliable on SLE10 though and can be compiled without conflicts. You need
to provide a source for a squashfs package and add it to the boot image
description in /usr/share/kiwi/image/isoboot/suse-SLES10/config.xml

You also need to add the following in this config.xml

 <drivers type="drivers">
   ...
   <file name="fs/squashfs/*"/>
 </drivers>

 <packages type="bootstrap">
   ...
   <package name="squashfs"/>
 </packages>
