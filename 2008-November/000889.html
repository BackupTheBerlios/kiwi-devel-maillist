> Does SLES 10 understand the opensusePattern? KIWI tells me, that the pattern is added, but KDE is missing.

Did you specify your image package section like this:

	<packages type="image" patternType="plusRecommended" patternPackageType="plusRecommended">
		...
	</packages>

If not you only get the required packages which is not enough
to run KDE smoothly


> I' ve some problems with the profiles:
> # kiwi -i /usr/share/kiwi/image/isoboot/suse-SLES10/
> Nov-21 21:27:07 <1> : Reading image description [ListProfiles]...
> Nov-21 21:27:07 <1> : default: [ Boot with default kernel ]
> Nov-21 21:27:07 <1> : smp: [ Boot with smp kernel ]
> # kiwi --create /usr/local/kiwi/mysles10 -d /usr/local/kiwi/mysles10-iso --add-profile default
> Nov-21 21:27:33 <1> : Reading image description [Create]...   failed
> Nov-21 21:27:33 <3> : Profile default: not found

the profile default is setup in the boot image description but you
don't have one in your system image description. There is no need to
specify --add-profile default it's the default anyway

If you want to setup a boot profile (select a profile for the boot image)
you need to add the attribute

   bootprofile="..."

as part of the <type >...</type> line in your system image config.xml
