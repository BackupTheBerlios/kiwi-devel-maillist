> I'm playing with kiwi this week and thank you for nice product to
> build OpenSUSE 10.3 LiveCD with Kiwi.
> 
> How if I want to build OpenSUSE 10.3 LiveDVD or Remastering DVD Iso
> (adding some madwifi driver just an example). Is there any existing
> config.xml or do I myself need to create another one ?

I'm sorry I don't have a DVD description available but you can
get to the point relatively fast if you make use of a pattern set

I use the following for testing:

<packages type="image" patternType="plusRecommended" patternPackageType="plusRecommended">
	<package name="kernel-default"/>
	<package name="bootsplash"/>
	<package name="bootsplash-theme-SuSE"/>
	<opensusePattern name="default"/>
	<opensusePattern name="x11"/>
</packages>

Hope that helps
