Hi Fruitlovers :-)

JFYI

The image descriptions and the kiwi code has been separated.
The description from the svn path:

	svn.berlios.de/svnroot/repos/kiwi/kiwi-head/system

has been moved to:

	svn.berlios.de/svnroot/repos/kiwi/images/kiwi-images-head

In principal there are two RPM package specs out now:

	svn.berlios.de/svnroot/repos/kiwi/
		--> the code and the boot parts

	svn.berlios.de/svnroot/repos/kiwi/images/
		--> the system image descriptions
	
Both specs are still combined in the openSUSE:Tools project.
So in terms of packages nothing will change. Packages are still
to be found here:

	http://ftp.opensuse.org/pub/opensuse/repositories/openSUSE:/Tools/
	
This has been done for the following reasons:

	1) descriptions may need repository tagging
	2) in order to provide kiwi as builder software the predefined
       descriptions must be handled outside
