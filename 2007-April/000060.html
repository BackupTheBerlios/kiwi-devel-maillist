Hi Fruit Lovers :)

I submitted v1.29 to openSUSE today. Most important changes are:

* virtual disks are no longer created with a 10GB default
  disk size but calculates the needed disk space including
  a 10% overhead.

* the netboot boot image support nfs-root mounted system
  images now. This means you can prepare the system image
  export it via a NFS server and let the boot image include
  the exported root fileystem. This is useful for diskless
  stations and should make integration of the ltsp easier.
  Details on how to setup NFS root images can be found in
  the documentation provided by the kiwi base package

	/usr/share/doc/packages/kiwi/kiwi.pdf

Have fun :-)
