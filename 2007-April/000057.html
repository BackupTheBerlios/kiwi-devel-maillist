Hi,

> I'm just starting to play a bit with KIWI and noticed that we treat both
> Required and Recommended in pattern files as Required.
> Is there a reason behind that? It sort of prevents me from creating a
> base install, from which I can expand the image further later on.

There is now a swtich available in config.xml to decide for

	patternType="onlyRequired"  --> only the required packages
	
	or

	patternType="plusRecommended"  ---> the required and recommended packages

default is "onlyRequired". To make use of it just setup the packages like
the following example shows:

	<packages type="image" patternType="onlyRequired">
		...
	</packages>
