Hello Community

Last week I posted a simple wrapper script that makes working with
kiwi little bit easy for users. The idea is to provide easy way to
build various images from already available descriptions. User does
not have to edit any config.xml file, all they have to work with is
one config file /etc/sysconfig/kiwi-build and one command
"kiwi-build-image" which takes just one switch for the image type to
build.

Here are the components of the package:

* Central configuration file /etc/sysconfig/kiwi-build:

Some variables that are in that file:

#(example,icecream,kde3,kde4,gnome,LimeJeOS)
#Set the imaging product, zypper install easy-kiwi-desc-<imagename> if
you do not have it installed
KIWI_PRODUCT="example"

#URL http://ftp5.gwdg.de/pub/opensuse/distribution/11.0/repo/oss/ to
build using internet repo
SUSE_INSTALL_SOURCE=

#Add space separated list of packages to be installed in the client
for use as localapps
ADDITIONAL_APPS=

* Functions file:
/usr/share/kiwi/build/kiwi-build-functions.sh

This file contains functions called by kiwi-build-image script.

* kiwi-build-image wrapper script:
kiwi-build-image has the following parameters;
   -p    - Perform a pre-check. This will check that mandatory
packages are installed and that the installation source can be
verified
   -cd   - Build: prepare, create KIWI Live CD/DVD system
   -cd1  - Prepare KIWI Live CD system
   -cd2  - Create KIWI Live CD system, to be run only if cd prepare
stage has been completed
   -u    - Build: prepare, create and deploy KIWI USB stick system
   -u1   - Prepare KIWI USB stick system
   -u2   - Create KIWI USB stick system, to be run only if cd/usb
prepare stage has been completed
   -u3   - Deploy KIWI USB stick system, to be run only if usb create
stage has been completed
   -xen  - Build: prepare and create Xen virtual machine image
   -vmx  - Build: prepare and create VMWare vmx image

Image descriptions that are slightly adapted to build all these image
types listed above are in separate packages, the available images are:
easy-kiwi-desc-LimeJeOS: Less than 100M openSUSE image by Hubert Mantel
easy-kiwi-desc-example: Minimal image with FVWM, firefox and Xterm
easy-kiwi-desc-gnome/kde4: Images used to create "official" live CDs from Coolo
easy-kiwi-desc-kde3: KDE3 image from Carlos Goncalves

Information on getting and using it are here:

http://dev.compiz-fusion.org/~cyberorg/2008/09/05/spin-opensuse-live-cd-or-usb-stick-image-easily/
