> This is a best guess attempt to show the example in 11.3 as well. I did
> not test this.

let us postpone this one for the following reasons:

1) the kernel on 11.3 will not work with the xen kernel at Amazon
2) we are currently in the evaluation phase for sle11 and Amazon
   Amazon will allow disk images types which means you can have your
   own kernel and initrd instead of registering a public kernel and
   initrd at Amazon. If we have an agreement here we need a special
   kernel called kernel-ec2 which we use to build the images.
   Most probably this kernel will be maintained for SLE only and
   our image examples needs to be changed then too.

For the moment the ec2 examples are more or less useless for
non Novell partners because we don't have a public kernel (aki)
and initrd (ari) image out there. With the change to a disk
image (vmx) we can upload our own kernel which still needs to
be compatible with Amazon but that's easier to achieve then
waiting for an official registration of a kernel and initrd
at Amazon
