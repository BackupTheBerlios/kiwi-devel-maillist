Hello kiwi lovers,

I've created a patch to KIWIImage.pm that enables the XEN image to run.
Together with the modifications that Marcus did to xenboot/suse-linuxrc
(also attached but in full), it is now possible to run kiwi created XEN
images on SLES10SP1RC1.

The patch was created against 1.21-25.1 that was on BS.

The only modification to suse-linuxrc is on this line;

	for rdevice in /dev/hda1 /dev/sda1 /dev/hda /dev/sda;do

Where /dev/hda and /dev/sda has been added.

Cheers,
Magnus
