Hi,

The attached patch does a few things, but the main point is too add
support for squashfs/unionfs images.  Using squashfs is straightforward
-- just set 'squashfs' as the image type.  Using unionfs is a little
trickier, and only netboot is supported currently.  You need to add
something like UNIONFS_CONFIG=/dev/hda2,/dev/hda3 to config.<MAC> where
the first partition is the writable one (where an ext2 filesystem will
be created), and the second is the read-only filesystem (squashfs,
cramfs, whatever).

This patch also allows you to specify "image" for the partition size in
config.xml, which makes the partition just large enough to fit the image
you've created.

James
