> > If i delete the kiwi.random directory from /tmp and rerun kiwi -p then kiwi -c it works:

kiwi extracts the kernel from the prepared image tree. After that it
doesn't exist in the prepared tree anymore. That happens because you need
the boot kernel outside of the boot image and it saves you a lot of space
in the boot image. If you remove the extracted kernel you are somehow
forced to regenerated the complete tree in order to get the kernel back
to the boot image.
