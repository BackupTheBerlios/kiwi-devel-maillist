Hi Fruit Lovers ;)

v1.39 is building. I'm sorry I didn't announce some release
steps in between to this one because it affects mainly Novell
internal changes and some work on the "installation source setup"
topic which is still work in progress and not ready for use

Beside this the 1.39 version includes a new feature for creating
the virtual disks of the "vmx" image type. It is now possible
to add the attribute:

	format="..."

to the <type> section of the config.xml. This allows you to
specify the virtual disk image format. kiwi always will create
a raw disk by default even if there is no format attribute
specified. Any other type needs to be specified. If you want
kiwi to create a VMWare image the attribute must be set to

	format="vmdk"

Supported formats are:

	vvfat
	vpc
	bochs
	dmg
	cloop
	vmdk
	qcow
	cow

Somehow I would love to see vhd (Microsoft virtual hard disk)
supported as well but right now there is no support for it.
