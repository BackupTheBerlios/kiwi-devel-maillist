> After looking over the code a couple of times to find out why I didn't get any bootsplash images, I noticed that:
>  
> a) Not everything from the cdboot catalog is copied to the boot catalog
> b) Nothing seems to be done to append a splash to the initrd (I might have overlooked something, but I don't think so)
>  
> Anyway... To get bootsplash working for me with various resolutions I copied the *.spl files from the openSUSE 10.2 DVD into cdboot (in suse-10.2 in this case) and made this change in KIWIImage.pm:
>  
>         #==========================================
>         # copy base CD files
>         #------------------------------------------
>         $kiwi -> info ("Setting up isolinux boot CD [$isoarch]");
>         qx (cp $gfx/* $destination);
>         qx (cp $CD/isolinux.cfg $destination);
>         qx (cp $CD/isolinux.msg $destination);
>         # MM: Inserted for bootsplash...
>         qx (cp $CD/*spl $destination);
>         $kiwi -> done ();
> I haven't done any checks on whether there actually *are* any *.spl files, but this works for me ;)
>  
> I'm sure it would be done much more elegantly, but since I seem to have seen many people wonder how to do it on IRC, I though I would post how I did it.

you are right with your concern. There is already a bug open for this problem:
