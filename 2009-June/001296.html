I think it would be nice if the <repository> element would support
multiple <source> elements. Therefore the following is possible:

<repository type="yast2">
    <source path="opensuse://11.1/repo/oss/"/>
    <source path="opensuse://11.1/repo/non-oss/">
</repository>

This allows grouping of all repositories of the same type into one block.
