separator which at that place is a bad idea. Attached is a patch
which should solve the problem.

- apply the patch with: 'patch -p0 < kiwi.patch' to your local
  installation.

- recreate the checksum: I assume you use 11.4. If not adapt
  the following command accordingly:

  kiwi --createhash /usr/share/kiwi/image/oemboot/suse-11.4

- rebuild the image
