Hi,

Finally getting some kiwi work in ;) . This patch adds a new attribute 
to the <repository> element called "imageinclude", a boolean. The code 
treats the attribute as having a "false" default value. When set to true 
(for using zypper package manager only at this point) the specified 
repository is added to the image being created, thus the repo is 
available to install additional packages or upgrade existing packages 
from within the running image.
