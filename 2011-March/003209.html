Hi,

The attached patch fixes 680148. The ISO generation code allows for 
either genisoimage or mkisofs. The checker does not support this. The 
patch addresses this problem and the checker no will be happy if either 
genisoimage or mkisofs is found.

Also changed the Iso code to use the locator rather than hard coding the 
path of the executables.

Patch is also attached to the bug report for completeness.
