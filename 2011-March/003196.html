Hi,

> Hey everyone,
> 
> kiwi.pl/main contains:
> 
> 	#==========================================
> 	# remove pre-defined smart channels
> 	#------------------------------------------
> 	if (glob ("/etc/smart/channels/*")) {
> 		qxx ( "rm -f /etc/smart/channels/*" );
> 	}
> 
> Even though I don't know what is/was accomplished by this, but I'd
> argue that no program should delete files in /etc except for the
> package manager when pruning packages.

well I do I agree but the problem with smart is that it adds
default source channels in /etc/smart/channels/ and always reads
them in which is bad when you want to use another source channel
set. I don't know if that behavior has changed, if yes I'm fine
to remove that hack

to my knowledge the "real" smart channels are in /var/lib/smart/
and kiwi also uses --data-dir for smart. But if there are channels
defined in /etc/smart/channels they are used in any case which
is a weird behavior
