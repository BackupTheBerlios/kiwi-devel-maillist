Hi,

due to the nature of the project I have moved kiwi from
svn to git to allow better handling of branches. The svn
repo has been switched to read-only mode so you can get
the kiwi source from here now:

* Anonymous Git Access via GIT
  git clone git://git.berlios.de/kiwi

* Developer GIT Access via SSH
  git clone ssh://developername@git.berlios.de/gitroot/kiwi
